# react-voice

React for Voice User Interface  
demo(Movie): [https://photos.app.goo.gl/N49a7h2rf617F94c8](https://photos.app.goo.gl/N49a7h2rf617F94c8)


`$ git clone https://github.com/hand-dot/react-voice.git`  
`$ cd react-voice`  
`$ npm i`  
`$ npm run ex`
